# Hospital Booking System

A responsive hospital booking and clinic management website developed using HTML, CSS, JavaScript, and Bootstrap. This project was created as a web development project to simulate a modern healthcare booking platform with multiple pages, appointment management interfaces, and user-friendly navigation.

## Features

- Responsive multi-page hospital website
- Appointment booking page
- Doctor and services pages
- Clinic information and reviews section
- Subscription/contact section
- Interactive navigation and UI components
- Mobile-friendly design using Bootstrap
- Custom CSS styling and JavaScript functionality

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Bootstrap 5
